# Wordle

Ce projet est le fruit d'une collaboration entre **Rémi Campistron**, **Théo Sarhane**, **DaSilva** Rafael, **Rafael Ducasse** et **Jonathan Maier**.  

Jeu de devinettes inspiré de **Wordle** : l’objectif est de trouver un mot de 5 lettres en un nombre limité d’essais, avec un retour visuel indiquant les lettres bien placées / présentes / absentes.

## Fonctionnalités

- Grille de tentatives (ex: 6 essais, à adapter)
- Validation du mot (dictionnaire / liste de mots)
- Feedback visuel (couleurs) sur chaque lettre
- Bouton rejouer 
- Responsive (mobile/desktop)

## Technologies
- HTML / CSS / JavaScript

## Installation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/DucasseRafael91/Wordle.git
   ```
2. Ouvrir `index.html` dans un navigateur
